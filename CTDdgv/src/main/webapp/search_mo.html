< Variant-orient(VL) >

1. In Variant-orient(VL) retrieval module, users can query the experimentally supported clinical function of ctDNA at Variant-Level by keyword combinations of disease type, gene symbol, variant type, and variant description (allowed to be filled in when the variants type is missense mutation).

2. Variant description section input format: G1202R, c.228C>T, chr3:g.37025749T>A; Multiple mutations should be separated by the enter key.

3. Quick match provide accurate search for diseases or genes, click 'add' to include them in your query list.

Note: options marked with * are required.
